# 外部资源

本目录下为终末地与森空岛内与Baker相关的UI资源

- **`avatar`** 头像 分辨率 156x156
    - `npc` npc头像
    - `operator` 干员头像
    - `others` 测试用和群组头像  
<br>

- **`bg`** 聊天框与头像及其他背景
<br>

- **`decorate`** 相关装饰性 UI 资源
<br>

- **`emoji`** emoji，但是有命名为`emoiji`的重复部分（？
<br>

- **`icon`** 图标
<br>

- **`image`** 剧情图片
<br>

- **`mask`** 遮罩
<br>

- **`others`** 文件命名中未明显体现所属类型
<br>

- **`sticker`** 大表情贴纸
    - game 游戏内贴纸，`png`格式，分辨率 168x168
    - skland 森空岛内贴纸，`webp`格式，分辨率 240x240
